Установка
-------------

Установка при помощи docker-compose:

    wget http://85.143.210.108:667/softomate.tar.gz
    tar xfz softomate.tar.gz
    cd softomate
    docker-compose up -d

Приложение будет слушать порт 667 и доступно по адресу http://localhost:667/api/v1/doc
